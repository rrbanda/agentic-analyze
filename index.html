Lyzr Control Plane -- Their Crown Jewel

"The Vercel for AI agents" -- accepts ANY framework, ANY cloud. This is what makes them sticky.

Agent Registry

Central catalog: framework, runtime, status, approval history, eval results. The "etcd" for agents.

Per-Agent Okta Identity

Dedicated Okta identity per agent. Scoped, attributed, auto-revoked on decommission.

Agent CI/CD

Git push → scan → RAI check → simulation eval → registry → staged promote. One-command rollback.

Simulation Engine

Persona × scenario matrix. 20+ parallel workers. Auto-hardening rewrites instructions on failure. CI/CD quality gate.

Cross-Framework

LangGraph, CrewAI, Agentforce, Copilot, custom -- one pipeline. No rewrite. Point at your repo.

Observability

Per-run: input, output, latency, tokens, cost, hallucination rate, tool accuracy.
