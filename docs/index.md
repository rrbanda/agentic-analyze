## Key Findings at a Glance

| Dimension | Lyzr's Position | Implication |
|-----------|----------------|-------------|
| **Category** | "Control Plane for AI Agents" -- framework-agnostic governance layer | Not competing with frameworks (LangGraph, CrewAI) -- governing them |
| **Funding** | $125M raised, $500M valuation (July 2026) | Well-capitalized to execute; 250% QoQ revenue growth |
| **Pricing** | $0.03--$0.30 per agent run (consumption-based) | Aligns cost with value; no seat taxes |
| **Customers** | Accenture, JPMorgan Chase, Pepsi, Willis Towers Watson, US Gov | Proven enterprise traction in regulated verticals |
| **Core Thesis** | Context engineering > model selection | Model-agnostic, cloud-agnostic, framework-agnostic |
| **Key Gap** | Governance layer for MCP/A2A protocols is still open | Strategic opportunity for whoever fills it first |
| **Market** | $10-12B standalone agentic AI (2026), 44-50% CAGR | Massive TAM but 89% of projects fail to reach production |
