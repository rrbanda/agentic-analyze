# Pre-Built Agents

## Overview

Lyzr offers 200+ pre-built agents across BFSI, healthcare, HR, sales, marketing, and customer support. These are not demos -- they have been through simulation and compliance testing. Three flagship products stand out:

---

## Amadeo -- The Banking Agentic OS

Amadeo is a multi-agent AI suite built specifically for banking. It automates core processes like customer onboarding, loan origination, KYC/AML, and regulatory reporting.

### Capabilities

| Process | What Amadeo Automates | Reported Results |
|---------|----------------------|-----------------|
| **KYC/AML** | Identity verification, sanctions screening, risk assessment, profile updates | 90% reduction in onboarding time; 80% fewer false positives |
| **Loan Origination** | Document collection, credit analysis, fraud checks, decisioning | 75% faster processing; 3x capacity increase |
| **Regulatory Compliance** | Monitors governing body sites, alerts on rule changes, suggests product updates | 30,000+ hours saved annually (large bank case study) |
| **Dispute Management** | Transaction analysis, evidence gathering, settlement drafting | Compressed 6-month LangChain build to weeks |
| **Customer Onboarding** | Document verification, credit check, compliance screening, account provisioning | 5x faster (5 days → minutes); 60% drop-off reduction |

### Headline Metrics

- **300% ROI** in year 1 (tracked across multiple implementations)
- **95% time savings** on manual workflows (KYC, underwriting, regulatory documentation)
- **60% reduction in compliance workload** (24 hours back per week per team)
- **~$1.02 per KYC run** (20+ agents, fully automated end-to-end)

### Architecture

Amadeo uses hierarchical multi-agent orchestration:

- Manager agent receives claims/requests
- Delegates to specialist agents (transaction analysis, evidence gathering, settlement drafting)
- Hallucination Manager keeps responses grounded in trusted data
- Lyzr Control Plane provides governance and audit layer
- Integrates via secure APIs with CRM, compliance software, loan management tools

## Why Pre-Built Agents Matter Competitively

Pre-built agents are a **go-to-market accelerator**:

1. **Time to value:** "Your KYC agent is live in 30 days" beats "Here's infrastructure, build your own."
2. **Proof of capability:** Showing a working banking agent is more convincing than a demo notebook.
3. **Land and expand:** Start with one vertical solution, expand to the full platform.
4. **Competitive moat:** Once Amadeo is processing KYC for a bank, switching costs are significant.

Infrastructure platforms that lack pre-built vertical solutions are competing on a different plane -- literally.
